Display galleries / slideshows of photos / videos in any post or page within your site.

== Description ==

This plugin allows you to create media galleries / slideshows containing videos and photos from sources such as YouTube and Vimeo.

Features Include:

- Include your photos, YouTube videos, and Vimeo videos.
- Optional media captions
- Ability to hide controls for YouTube videos
- Ability to re-order media within a slideshow / gallery

== Installation ==

1. Unzip 'mediaview.zip' and upload `mediaview.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin (MediaView) through the 'Plugins' menu in WordPress.
3. Create a shortcode and copy shortcode onto a page or post.
4. Add media for gallery / slideshow just created.
6. Enjoy!
